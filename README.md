# RaceTimer

Static web app for race start timing and line setup.

## Run locally

Open `index.html` in a browser, or serve the folder with any static server.

## Files

- `index.html` – main UI
- `app.js` – logic
- `style.css` – styles
- `map.html` / `map.js` – map view
- `vendor/` – Leaflet assets
- `manifest.json` / `sw.js` – PWA assets
